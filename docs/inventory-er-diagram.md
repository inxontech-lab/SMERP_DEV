# Inventory Schema ER Diagram

Downloadable PDF files for the inventory ER diagram:

- `Inventory-ER-Diagram.pdf` (repo root, easy download)
- `docs/inventory-er-diagram.pdf` (docs copy)

Both files are generated from the inventory entity models and relationships in `Domain/SaasDBModels`.

To regenerate the PDFs:

```bash
python docs/generate_inventory_er_pdf.py
```
